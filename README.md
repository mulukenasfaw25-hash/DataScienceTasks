📌 Level 1: Prime Number Checker
🔍 Description

This program checks whether a given number is a prime number.

⚙️ How It Works
Numbers ≤ 1 are not prime
Checks divisibility from 2 to √n
Efficient and optimized approach
▶️ Usage
python level1.py
💻 Example
enter a number: 11
prime number
📌 Level 2: Vowel Counter
🔍 Description

Counts the number of vowels in a given string.

⚙️ How It Works
Iterates through each character
Matches against aeiouAEIOU
Counts total vowels
▶️ Usage
python level2.py
💻 Example
enter a string: Programming
number of vowels: 3
📌 Level 3: Tic-Tac-Toe Game (CLI)
🔍 Description

A tic-tac-toe game between two users

🎮 Features
✅ Two-player gameplay (X vs O)
✅ Input validation (no crashes on wrong input)
✅ Prevents overwriting moves
✅ Clean and readable board display
✅ Win detection system
✅ Draw detection
✅ Replay option
▶️ Usage
python level3.py
